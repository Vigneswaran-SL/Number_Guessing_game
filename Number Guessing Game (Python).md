 Number Guessing Game (Python)

A simple command-line Number Guessing Game built using Python. The program generates a random number between 1 and 50, and the player has 5 attempts to guess it correctly.

 Features

-  Random number generation
-  User guesses a number between 1 and 50
-  Input validation using "try" and "except"
-  Hint system ("Too High" / "Too Low")
-  Limited to 5 attempts
-  Winning and losing messages

 Technologies Used

- Python 3
- "random" module
- PyCharm

 How to Run

1. Make sure Python 3 is installed.
2. Clone this repository:

git clone https://github.com/Vigneswaran-SL/number-guessing-game.git

3. Open the project folder.

4. Run the program:

python guessing_game.py
